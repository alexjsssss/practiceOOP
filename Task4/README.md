# Лабораторна робота №4  
**Тема:** Використання шаблону Factory Method. Табличне відображення результатів. Поліморфізм.  

---

## 🎯 Мета роботи
1. Використати вихідний код попередньої лабораторної роботи та застосувати шаблон **Factory Method (Virtual Constructor)** для створення похідних класів, що відображають результати у вигляді текстової таблиці з параметрами, які задає користувач.  

2. Продемонструвати:
   - перевизначення (**overriding**),  
   - перевантаження (**overloading**),  
   - поліморфізм (**dynamic method dispatch**).  

3. Реалізувати діалоговий інтерфейс із користувачем.  

4. Створити клас для тестування основної функціональності.  

5. Використати коментарі для автоматичної генерації документації засобами **javadoc**.
