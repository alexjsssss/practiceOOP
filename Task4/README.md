# Лабораторна робота №4  
**Тема:** Використання шаблону Factory Method. Табличне відображення результатів. Поліморфізм.  

---

## 🎯 Мета роботи
1. Використати вихідний код попередньої лабораторної роботи та застосувати шаблон **Factory Method (Virtual Constructor)** для створення похідних класів, що відображають результати у вигляді текстової таблиці з параметрами, які задає користувач.  

2. Продемонструвати:
   - перевизначення (**overriding**),  
   - перевантаження (**overloading**),  
   - поліморфізм (**dynamic method dispatch**).  

3. Реалізувати діалоговий інтерфейс із користувачем.  

4. Створити клас для тестування основної функціональності.  

5. Використати коментарі для автоматичної генерації документації засобами **javadoc**.

## ⚙️ Опис класів
- **Item2d** – модель даних (маса, висота, енергія, max1). Реалізує `Serializable`.  
- **Calc** – виконує обчислення потенційної енергії та пошук максимальної кількості послідовних `1`.  
- **View** – інтерфейс для відображення результатів (методи `viewInit`, `viewShow`, `viewSave`, `viewRestore`).  
- **ViewResult** – конкретна реалізація інтерфейсу `View`. Працює з колекцією `ArrayList<Item2d>`.  
- **Viewable** – інтерфейс фабричного методу.  
- **ViewableResult** – фабрика, яка створює об’єкт `ViewResult`.  
- **ViewTable** – клас для табличного відображення результатів (колонки m, h, E, max1).  
- **ViewableTable** – фабрика, яка створює об’єкт `ViewTable`.  
- **Main / MainTest** – точка входу та тестування: генерація, серіалізація та відновлення даних.

## ✅ Висновки

- Реалізовано серіалізацію та відновлення стану об’єктів.
