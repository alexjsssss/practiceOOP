# Серіалізація та відновлення стану об'єкта

## Опис завдання
У цьому завданні необхідно:
1. Розробити клас для демонстрації збереження та відновлення стану об'єкта за допомогою **серіалізації**.
2. Показати особливості використання **transient** полів (які не серіалізуються).
3. Продемонструвати роботу програми у діалоговому режимі.
