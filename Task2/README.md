# Лабораторна робота №2  
**Тема:** Серіалізація та відновлення стану об’єкта. Використання `transient` полів.  

---

## 🎯 Мета роботи
1. Розробити клас для демонстрації збереження та відновлення стану об’єкта за допомогою серіалізації.  
2. Показати особливості використання `transient` полів (які не серіалізуються).  
3. Продемонструвати роботу програми у діалоговому режимі.

## ⚙️ Опис класів
- **Item2d** – клас даних, який реалізує `Serializable`. Має поля для збереження результатів обчислень.  
- **Calc** – виконує обчислення потенційної енергії та знаходить максимальну кількість послідовних `1` у двійковому представленні.  
- **Main** – демонструє процес серіалізації та десеріалізації у діалоговому режимі.  
- **MainTest** – тестовий клас, який перевіряє коректність роботи програми.
