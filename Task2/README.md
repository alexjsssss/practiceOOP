# Серіалізація та відновлення стану об'єкта

## Опис завдання
У цьому завданні необхідно:
1. Розробити клас для демонстрації збереження та відновлення стану об'єкта за допомогою **серіалізації**.
2. Показати особливості використання **transient** полів (які не серіалізуються).
3. Продемонструвати роботу програми у діалоговому режимі.

---

## Опис
- Використовується клас `Item2d`, який реалізує інтерфейс `Serializable`.
- Поля, позначені як `transient`, не потрапляють у серіалізований файл.
- Клас `Calc` виконує обчислення та агрегує дані у `Item2d`.
- Клас `Main` демонструє процес серіалізації та десеріалізації.
- Клас `MainTest` перевіряє коректність роботи програми.
