# Лабораторна робота №3  
**Тема:** Використання шаблону Factory Method. Серіалізація та відновлення колекції об’єктів.  

---

## 🎯 Мета роботи
1. Використати вихідний текст попередньої лабораторної роботи як основу.  
2. Забезпечити розміщення результатів обчислень у колекції з можливістю збереження/відновлення.  
3. Реалізувати шаблон проектування **Factory Method (Virtual Constructor)**.  
4. Розширити ієрархію інтерфейсом "фабрикованих" об’єктів для відображення результатів.  
5. Реалізувати методи відображення у текстовому вигляді.  
6. Продемонструвати роботу програми та перевірити коректність виконання.

## ⚙️ Опис класів
- **Item2d** – модель даних (маса, висота, енергія, maxOnes). Реалізує `Serializable`.  
- **Calc** – виконує обчислення потенційної енергії та пошук максимальної кількості послідовних `1`.  
- **View** – інтерфейс для відображення результатів (методи `viewInit`, `viewShow`, `viewSave`, `viewRestore`).  
- **ViewResult** – конкретна реалізація інтерфейсу `View`. Працює з колекцією `ArrayList<Item2d>`.  
- **Viewable** – інтерфейс фабричного методу.  
- **ViewableResult** – фабрика, яка створює об’єкт `ViewResult`.  
- **Main/MainTest** – точка входу, демонструє генерацію, серіалізацію та відновлення даних.
